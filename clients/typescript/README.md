# Metorial Function Bay Client

A client library for interacting with the Metorial Function Bay platform using TypeScript.

## Usage 

```ts
import { createSignalClient } from '@lowerdeck/forge-client';

let forge = createSignalClient({
  endpoint: '...'
});
```
